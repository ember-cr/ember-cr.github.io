🕹️ How It Works

Getting started with Ember is simple and fast.

1

Sign In

Create an account or sign in to get started. Google sign-in coming soon.

2

Create or Join

Create a room with custom settings, or use a 6-digit invite code to join one.

3

Start Chatting

Send messages, share files, and see who's online with status indicators.

4

Customize

Pick a theme, adjust sound settings, and personalize your profile anytime.

Standalone Mode Toggle

Manual toggle in Settings — runs entirely without Supabase. All data saved to LocalStorage. Cross-tab real-time chat via BroadcastChannel.

Bug Fixes & UX

Fixed blank screen on Explore Public Chats. Improved public mode messaging and hardened loading logic.

Public Rooms & Explore

Rooms can now be made public and discoverable. New Explore Public Chats page. Replaced static Global Chat with community-driven public rooms.

Sound System

Initial Standalone Mode — auto-activates when Supabase env vars are missing. LocalStorage + BroadcastChannel.
